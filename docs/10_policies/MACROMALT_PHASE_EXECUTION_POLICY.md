# MACROMALT_PHASE_EXECUTION_POLICY

작성일: 2026-03-17
적용 범위: macromalt 프로젝트 전체 Phase 공통 규칙

---

## 1. 실출력 검증 보고서

- Phase 진행마다 실출력 검증 보고서를 작성한다.
- 저장 위치: `docs/88_etc/REPORT/`
- 파일명 형식: `REPORT_PHASE{번호}{서픽스}_실행내용.md`

---

## 2. 커밋 & 핸드오프

- Phase 진행마다 커밋을 생성한다.
- 커밋과 함께 해당 Phase의 핸드오프 문서를 작성한다.
- 핸드오프 저장 위치: `docs/88_etc/Handoff/`
- Phase 최종 종료 시 총괄 핸드오프를 별도로 작성한다.

---

## 3. 미세 조정 횟수 제한

- Phase당 미세 조정(서픽스 A~E 등)은 최대 **5회**까지 허용한다.

---

## 4. 반복 문제 발생 시 Phase 승격

- 동일한 문제가 미세 조정 내에서 반복적으로 발생하면 해당 문제를 **다음 Phase로 승격**하여 처리한다.
- 승격된 Phase는 독립된 목표와 검증 기준을 가진다.
