# MACROMALT SLOT POLICY

기준일: 2026-03-16

## 1. 슬롯 정의

macromalt는 발행 시간대에 따라 아래 슬롯을 사용한다.

- `morning`
- `evening`
- `us_open`
- `default`

목표:
- 같은 테마라도 시간대별 질문 구조를 다르게 만든다
- 반복 테마가 있더라도 반복 관점을 완화한다

---

## 2. 슬롯별 시간 정책 (KST 기준)

### morning
- 대략 05:00 ~ 11:59
- 한국장 개장 전 문맥 포함

### evening
- 대략 14:00 ~ 20:59
- 한국장 마감 리뷰 문맥

### us_open
- 미국장 개장 30분 전 중심
- **DST**: 21:30 ~ 22:29
- **non-DST**: 22:30 ~ 23:29

### default
- 위 슬롯에 속하지 않는 예외/수동 실행용 중립 슬롯
- 특정 시장 개장/마감 문맥 강제 없음

---

## 3. 슬롯별 목적

### morning
- 미국장 마감 요약
- overnight 변수 분석
- 한국장 개장 전 체크포인트

### evening
- 한국장 실제 결과
- 오전 예상과 실제의 차이
- 내일 체크포인트

### us_open
- 오늘 밤 미국장 핵심 변수
- 뉴스/선물/금리/유가/실적 체크
- 반영/되돌림 시나리오

### default
- 현재 시점 기준 일반 브리핑
- 특정 개장/마감 문맥 강제 없음

---

## 4. 슬롯별 Post1 / Post2 역할

### morning
- **Post1**: 미국장 마감 원인 → 한국장 프리뷰
- **Post2**: 한국장 개장 전 체크할 종목/섹터

### evening
- **Post1**: 오늘 한국장 실제 결과 → 내일 변수
- **Post2**: 오늘 의미 있었던 종목/섹터 → 내일 관전 포인트

### us_open
- **Post1**: 오늘 밤 미국장 변수 브리핑
- **Post2**: 미국장에 상대적으로 민감한 종목/섹터

### default
- **Post1**: 현재 시점 핵심 변수 일반 분석
- **Post2**: 현재 시점 민감 종목/섹터 일반 브리핑

---

## 5. 중복 완화 원칙

- 같은 테마 자체는 시장이 강하게 지배할 때 허용 가능
- 대신 같은 테마 + 같은 질문 구조 + 같은 종목군 반복을 줄인다
- 최근 발행 이력을 활용해 관점 회전 지시를 넣는다
- 슬롯이 다르면 같은 theme라도 허용 폭을 넓힌다

---

## 6. 운영 정책

### 현재 테스트 단계
- 시간 기반 감지 허용
- 수동 실행에서도 slot 인식 가능

### 자동화 이후 권장
- 스케줄러가 slot 값을 명시 전달
- `default`는 예외/복구/수동 실행 전용
- `slot=default` 발생은 운영 로그에 명확히 남긴다

이 문서는 macromalt의 발행 시간대 분기 정책 문서다.
