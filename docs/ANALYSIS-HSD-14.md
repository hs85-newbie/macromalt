## 2. 상세 분석

### 2.1 주입 경로 #1 — Gemini Reviser 재주입 (핵심 원인)

- **파일**: `generator.py`
- **위치**: `GEMINI_REVISER_SYSTEM` (5754, 5760, 5763행)
- **현재 코드**:
  ```
  # 5754행
  - [해석] / [전망] 태그 누락된 해석·전망 문장에 태그 추가 (삭제 금지, 추가만)

  # 5760행
  1. [해석]/[전망] 태그 누락된 해석·전망 문장에 태그 추가

  # 5763행
  - "[전망] 2026년 X는 Y로 예상된다(출처 추정)" 형태로 재서술
  ✅ 수정 후: "[전망] 삼성전자의 2026년 매출은 300조 원으로 추정된다(미래에셋 리서치)"
  ```
- **작동 방식**: GPT Writer가 태그 없이 생성 → Reviser가 "누락"으로 판단 → 강제 주입
- **영향**: 모든 게시글에서 Reviser 단계를 거치는 한 100% 재발
