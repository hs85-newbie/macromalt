# step03-07. QA Checklist (Step03)

Step03의 최종 만족 여부를 판단하는 점검표입니다.

## 1. Trust Surface Verification (Must-Fix)
- [ ] **About Page**: 확정된 5단계 섹션 스켈레톤(The Mission, Methodology, Coverage Area, Continuity, Contact)이 모두 노출되는가?
- [ ] **Disclosure Box**: 오직 `single` 포스트 상단에만 `DISCLOSURE & METHODOLOGY` 박스가 정상 노출되는가?
- [ ] **Disclosure Exclusion Scope**: Home, About, Category, Archive, Search, Static Pages에서 고지 박스가 **제외**되었는지 확인했는가?
- [ ] **Footer Append**: 기존 사이트 제목 하단에 단일 저작권/정체성 레이어가 정상 추가되었는가? (중복 레이어 발생 금지)

## 2. Discovery & Utility Verification
- [ ] **Search Header**: 검색 시 `SEARCHING THE MACROMALT ARCHIVE:` 레이블이 노출되는가?
- [ ] **Meta Consistency**: 모든 목록 디자인에서 날짜, 작성자, 태그가 Mono-type 폰트로 일관성 있게 표시되는는가?
- [ ] **Read More**: 모든 유도 링크가 `FULL ANALYSIS →`로 일치화되었는가?

## 3. Reporting Standards
- [ ] **08A~08E**: 모든 리포트 문서에서 모순된 상태값(Pending Sync vs Deployed)이 발견되지 않는가?
- [ ] **Live Proof**: 실제 라이브 사이트의 데스크탑 및 모바일 캡처본이 포함되었는가?
