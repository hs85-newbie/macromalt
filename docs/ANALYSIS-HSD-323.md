### 2-1. 이슈 ① — `[전망]` 태그 잔존 (HSD-7 부분 반영의 결과)

#### 배경

2026-03-27 커밋 `0cdc8db`에서 GPT 라이터 프롬프트만 자연어 표지로 전환했고, HSD-7 분석(2026-04-08)은 Gemini Reviser·품질 게이트·후처리 계층을 모두 고쳐야 한다고 지적했다. 그러나 현재 코드 상태를 점검한 결과 **HSD-7의 P0 수정은 부분 반영**됐고 다음 4개 위치에서 `[전망]`이 여전히 재주입되거나 잔존한다.

#### 잔존 위치 #1 — 품질 게이트 #6이 여전히 태그 존재를 합격 조건으로 요구

- **라인**: 7950~7951

- **문제**: GPT가 자연어 표지(`~로 예상된다`, `~할 것으로 전망된다`)로만 작성하더라도, 본문에 "확실하지 않"이라는 표현이 없으면 게이트 #6 FAIL 처리되어 Reviser가 강제로 태그를 다시 주입하도록 유도한다.

### 2-2. 이슈 ② — 참고출처 중복 기재 + 날짜불명 노출

#### 핵심 함수: `_format_source_section()._rebuild()` (4760~4792행)

```python
def _rebuild(match: re.Match) -> str:
    raw = match.group(0)
    raw = re.sub(r"<br\s*/?>", "\n", raw, flags=re.IGNORECASE)
    raw = re.sub(r"</li>|</p>|</div>", "\n", raw, flags=re.IGNORECASE)
    raw = re.sub(r"<[^>]+>", "", raw)
    parts = re.split(r"[\n·•]+", raw)
    items = [p.strip(" \t\r-") for p in parts if len(p.strip()) > 2]
    items = [i for i in items if i.lower() not in ("참고 출처", "참고출처")]

    brokers = [i for i in items if any(k in i for k in BROKER_KW)]
    news    = [i for i in items if any(k in i for k in NEWS_KW) and i not in brokers]
    others  = [i for i in items if i not in brokers and i not in news]
    ...
```

#### 누락 #1 — 중복 제거 로직 없음

- 파서가 항목을 분리한 후 **중복(set) 제거 단계가 없다**. GPT가 동일 출처를 여러 번 나열하면 그대로 출력된다.
- 정규화 키(예: "출처명 + 날짜") 기준 dedup이 필요하지만 미구현.

#### 누락 #2 — 날짜불명 필터링 없음

- 파서는 항목 길이만 검사(`len(p.strip()) > 2`)하며, 날짜 패턴(`YYYY.MM` / `YYYY.MM.DD` / `YYYY-MM` 등)이 없는 항목도 출력에 포함된다.
- 본문 데이터 수집 단계에서는 `date_tier = "unknown"`(라벨 `[날짜불명]`)을 분류하지만(`generator.py` 4242~4246행), 참고 출처 섹션의 최종 렌더링은 이 정보를 잃어버린 상태로 GPT가 자유 작성한 텍스트만 가지고 수행하기 때문에 날짜불명 여부를 알 수 없다.

#### 누락 #3 — GPT 프롬프트가 중복 금지·날짜불명 제외를 명시하지 않음

- **파일**: `generator.py`
- **라인**: 5419~5421 (Post1 프롬프트), 5587~5589 (Post2 프롬프트)

```
8. 참고 출처 섹션: <h3>참고 출처</h3> 다음에 <ul>/<li>로 각 출처를 한 줄씩 나열.
   형식: <li>출처명, 날짜</li> (예: <li>매일경제, 2026.04.08</li>)
   각 출처를 반드시 별도 <li> 태그에 작성할 것. 쉼표로 여러 항목을 한 줄에 나열 금지.
```

- HSD-8에서 분리 버그를 고치며 추가된 형식 명시는 있으나, **"동일 출처+날짜는 1회만"** / **"날짜를 알 수 없는 출처는 기재 금지"** 조항이 누락됐다.

#### 누락 #2 — GPT Writer 단계: 본문 내 인용 출처 개수 검사 부재
