## 6. Cross-Sample Diagnosis

### 잔존 시스템 병목 2가지

**병목 1 — "파악됩니다" / "추정됩니다" + `[전망]` 사각지대**

GPT가 `늘어날 것으로 전망됩니다` 대신 `파악됩니다`, `증가한 것으로 추정됩니다`를 선택할 때:
- Phase 15D: "파악됩니다"가 `_P15D_CONFIRMED_VERB_MARKERS`에 없어 제거 미발동
- Phase 15E: "파악됩니다"가 `_P15E_FORECAST_VERB_MARKERS`에 없어 탐지 미발동
- 결과: `[전망]` 태그 그대로 노출

**병목 2 — "한 것으로 추정됩니다" regex 미커버**

"증가한 것으로 추정됩니다" — `것으로 추정됩니다`를 Phase 15E가 forecast verb로 탐지하지만, `([가-힣]+)할` / `([가-힣]+)날` regex에 "한"이 없어 교정 불가. "추정됩니다" 단독 제거 로직 미비.

### 근본 원인

Phase 15D와 Phase 15E는 모두 **특정 동사 형태**에 의존한다. GPT는 이 동사 형태를 피해 다른 표현을 선택하므로, 교정이 발동되지 않는 채 `[전망]` 태그만 잔존하는 상황이 반복된다.

해결 방향: `[전망]` + `{run_year}년 {past_month}월` 조합이 탐지되면 동사 형태에 무관하게 태그를 제거하는 월 기반 강제 제거 로직 필요.
