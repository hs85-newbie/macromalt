## 1. ROI 시뮬레이션

### 1-1. 전제 조건

**BYOK(Bring Your Own Key) 모델 기준** — 사용자가 OpenAI/Gemini 키 직접 입력, API 비용은 사용자 부담.
macromalt 운영자 월 고정비용만 계산.

| 항목 | 월 비용 | 비고 |
|------|---------|------|
| FastAPI 서버 (Railway/Render) | ₩14,000~28,000 | 초기: Starter plan $10 |
| 도메인/SSL | **₩3,200** | **10년 선납 ₩382,950 기준 (월 환산)** |
| Toss Payments 수수료 | 매출의 3.3% | 건당 수수료 |
| **월 고정 현금 지출** | **≈ ₩17,200** | 수수료 제외 (서버 Starter 기준) |

> ✅ **실비 보정**: 도메인/서버 비용 10년 계약 선납 완료 (₩382,950/10년 = ₩3,200/월).
> 사실상 서버를 무료 티어(Render free/Railway hobby) 사용 시 **월 고정비 ≈ ₩3,200** 수준으로 수렴.
> 손익분기점은 첫 구독자 1명 결제 즉시 달성.

**구독 가격 (제안안)**
- Starter: ₩49,900/월 → 수수료 후 실수령 ₩48,253
- Pro: ₩99,900/월 → 수수료 후 실수령 ₩96,603

## 2. 경쟁사/유사 서비스 분석

### 2-1. 직접 경쟁사

**결론: 전 세계적으로 직접 경쟁사 없음**

"AI 투자/주식 분석 → 콘텐츠 생성 → WordPress 자동 발행"을 통합한 전문 SaaS 또는 플러그인은 2026년 3월 현재 존재하지 않음.

| 서비스 | 유형 | 한계 |
|--------|------|------|
| PRISM-INSIGHT | 오픈소스 주식 분석 에이전트 | WordPress 발행 없음, SaaS 아님 |
| EODHD WordPress Plugin | 주식 데이터 위젯 | 콘텐츠 생성 없음 |
| 씽크풀 | AI 금융 정보 서비스 | 독립 플랫폼, 개인 블로그 연동 없음 |
| 가제트AI (한국) | AI 블로그 자동화 | 금융 특화 없음 |
| 다마고치 (한국) | AI 블로그 자동화 | 쿠팡/애드센스 특화, 투자 분석 없음 |

### 3-2. 가격 적정성 판단

**Starter ₩49,900 → 적정**
- 범용 AI 콘텐츠 도구($49/월)와 동일 가격대
- 금융 특화 + 검증 파이프라인 = 프리미엄 정당화 가능
- BYOK로 API 비용 분리 → 실질 마진 확보

**Pro ₩99,900 → 적정 (단, 가치 전달이 관건)**
- 경쟁사 대비 2배 가격이지만 기능 격차가 훨씬 큼 (5테마 멀티파이프라인, 21개/일)
- 포스팅당 ₩284 → 외주 대비 35~176배 절감 = 가치 명확
- 단, "왜 Pro가 Starter의 2배인가"를 랜딩 페이지에서 명확히 설명해야 함

## 4. 종합 결론

### 진행 권장 여부: ✅ 권장

**근거:**
1. **공백 시장** — 직접 경쟁사 없음. 선점 기회
2. **ROI 빠름** — 구독자 5명에서 7개월 내 개발비 회수 가능
3. **가격 적정** — 시장 벤치마크 대비 합리적, 프리미엄 정당화 가능
4. **사용자 가치 명확** — 포스팅당 ₩284 vs 외주 ₩10,000~₩50,000

**핵심 전제 조건:**
- BYOK 모델 유지 (API 번들 시 단가 역전)
- Free 티어로 수요 검증 먼저 (빠른 출시 후 전환율 측정)
- Pro 가치 제안 랜딩 페이지에서 명확히 설명 필요
